## ğŸš€ Getting Started

### Prerequisites

- Node.js >= 18.0.0 and <= 22.x.x
- npm >= 6.0.0

### Installation

```bash
npm install
```

### Development

Start your Strapi application with autoReload enabled:

```bash
npm run develop
# or
yarn develop
```

### Production

Start your Strapi application with autoReload disabled:

```bash
npm run start
# or
yarn start
```

### Build

Build your admin panel:

```bash
npm run build
# or
yarn build
```

### Database Seeding

To seed the database with example data:

```bash
npm run seed:example
# or
yarn seed:example
```
